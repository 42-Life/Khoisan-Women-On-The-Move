## Khoisan Women On the Move
This project is a web app outlining the activities of Khoisan Women On the Move, a Cape Town based South African NGO. KSWOTM's work centres sustainable community development, community-based mentorship, and the alleviation of gender-based violence in Cape Town. 

This web app was developed by myself, Ajani Hickling, on a volunteer basis.
